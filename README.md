# 房产知识模拟考试系统

一个基于 Web 的在线考试系统，支持单选、多选和判断题。

## 功能特点

- 📝 支持多种题型（单选、多选、判断题）
- 🎯 左侧答题卡，可快速跳转到任意题目
- ✅ 即时答题反馈和详细解析
- 📊 自动计分功能
- 💾 答题状态实时保存
- 📱 响应式设计，支持各种屏幕尺寸

## 在线访问

访问 [GitHub Pages 链接]（部署后填写）

## 本地运行

1. 克隆本仓库
```bash
git clone [仓库地址]
```

2. 使用任意 HTTP 服务器运行
```bash
# 使用 Python
python3 -m http.server 8080

# 或者使用 Node.js
npx http-server
```

3. 在浏览器中打开 `http://localhost:8080`

## 技术栈

- HTML5
- CSS3
- 原生 JavaScript
- 无需任何框架或依赖

## 项目结构

```
├── index.html       # 主页面
├── style.css        # 样式文件
├── script.js        # 交互逻辑
├── data.js          # 题目数据
└── README.md        # 说明文档
```

## 使用说明

1. 点击左侧答题卡上的题号可快速跳转
2. 选择答案后点击"确认答案"查看结果
3. 完成所有题目后点击右上角"交卷查看成绩"
4. 支持键盘左右方向键切换题目
